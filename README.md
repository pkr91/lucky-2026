# 🔮 Lucky Universe 2026
## 2026 럭키 유니버스 — 한양대학교 인간인공지능 수업 팀 프로젝트
본 프로젝트는 한양대학교 산업융합학부 인간-인공지능 협업 제품 서비스 설계 수업(2025년 가을학기)의 기말 프로젝트 활동으로 진행된 결과물입니다. 본 수업의 지도 교수는 한양대 산업융합학부 정철현 교수(inbass@hanyang.ac.kr) 입니다. 코드와 문서는 오픈소스(MIT 라이센스)이므로 자유롭게 참조/사용하시되 사용으로 인한 모든 리스크는 스스로 감당하셔야 합니다.

## 📖 프로젝트 소개 (About Project)

AI 기반 초개인화 행운 생성 데모 웹앱

“오늘의 운세가 심심하다고? 이제 나만의 행운 유니버스를 만들어보자!”

2026 Lucky Universe는 생성형 AI(Google Gemini 2.5 Flash)를 기반으로
2026 종합 운세 → 오늘의 운세 & 미션 → 사랑운 → 직업/재물 리포트 → 4대 운세 분석 → 빌런 경보 → 대박 캘린더 → AI Pixel 캐릭터 생성
순서로 이어지는 초개인화 Luck Experience Web App입니다.

단순 운세 사이트가 아니라,
🔥 당신의 2026년 세계관을 생성하는 AI 콘텐츠 플랫폼입니다.

---

## 🎯 기획 의도

- **초개인화 경험:** MBTI + 사주 기반 운세 제공  
- **AI 인터랙션:** Pixel Art ‘럭키 다마고치’ 생성  
- **데일리 재미 요소:** 행운 점수 / 미션 / 로또 번호  
- **모바일 퍼스트 UI 디자인**

---

## ✨ 주요 기능 (Features)

### 🔮 1. 2026 나의 운세

앱을 실행하면 가장 먼저 **올해 전체 흐름(총운)**을 보여줍니다.

올해의 주요 테마

기운의 성향(도전·안정·성장 등)

2026 운세 전체 스토리의 프롤로그 역할

### 🎮 2. 오늘의 운세 & 데일리 미션

2026년 총운 다음에는 오늘의 행운 콘텐츠가 이어집니다.

🎯 오늘의 행운 점수

0~100점으로 당일의 기운을 정량화.

✨ 행운의 초성

오늘의 의미를 상징하는 초성 힌트 제공.

🏆 데일리 미션

오늘 운세 기반 행동 추천 예시:

파란색 아이템 착용

10분 산책

책상 정리하기

목표 회상하기

### 💘 3. 오늘 내 사랑 반쪽 찾기

AI가 MBTI·성향을 분석해
오늘 가장 궁합이 좋은 파트너 타입을 추천합니다.

오늘의 운세에 맞는 궁합 MBTI

### 💰 4. 직업 & 재물 리포트

2026년 전체 흐름 + 오늘의 기운을 기반으로
현실적인 분석 리포트를 제공합니다.

2026년 예상수입

나의 업무 스타일

숨겨진 재능 발견

추천 천직 Best 3

### 📜 5. 2026 4대 운세 심층 분석

AI가 올해의 핵심 영역 4가지를 세부적으로 분석합니다.

💰 재물운

❤️ 애정운

🧠 직업운

💪 건강운

각 운세는 “근거 + 해석 + 예측 포인트” 형태로 제공됩니다.

### ⚠️ 6. 2026 빌런 경보

### 🗓️ 7. 대박 캘린더 (황금 말발굽일)

2026년 중 운세 상승 구간을 추천해줍니다.

행운 상승일


### 👾 8. 럭키 다마고치 생성

마지막으로 세상에 단 하나뿐인 Pixel Mascot을 생성합니다.

생성 기준:

소원 → 속성(Element)

2026 운세 흐름 → 외형/컨셉 반영

이 캐릭터는 사용자의 2026년 세계관을 상징하는
AI Pixel Companion입니다.
